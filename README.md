# Обучение

## Урок 1. Освоение Git

### Команды:

1. git init
2. git log
3. git push
4. git log --oneline
5. git revert HEAD - делает отмену коммита и делает коммит отмены
6. git reset --hard 'hash-commit' - убирает коммит из ветки до указанного коммита

Полный курс Git: https://githowto.com/ru/setup
